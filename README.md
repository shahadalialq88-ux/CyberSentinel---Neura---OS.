# 🛡️ CyberSentinel | Forensic Neural OS (v3.0)

*CyberSentinel* is a high-performance security engine built for real-time URL forensic analysis. By integrating *Shannon Entropy* mathematical modeling with heuristic pattern detection, it provides instant threat intelligence and privacy-focused security assessments.

---

## 🔬 The Science Behind the Engine
Unlike traditional scanners that rely on static blacklists, CyberSentinel uses:
- *Neural Heuristics:* Analyzes URL structure for social engineering triggers and phishing patterns.
- *Shannon Entropy Analysis:* Quantifies the randomness of a domain to detect *DGA (Domain Generation Algorithms)* often used by malware and Command & Control (C2) servers.
- *Risk Scoring Algorithm:* A multi-layered scoring system (0-15+) that evaluates protocol safety, obfuscation, and keyword patterns.

## 🛠️ Core Features
- *Real-Time Forensic Scanning:* Instant detection of malicious patterns and high-risk domains.
- *Automated PDF Reporting:* One-click generation of professional forensic logs, including risk breakdowns and analyst attribution.
- *Privacy First:* Zero-data transmission; all calculations are performed locally, ensuring data sovereignty.
- *Dynamic Risk Visualization:* Color-coded status alerts (SECURE ✅, WARNING ⚠️, CRITICAL 🚨).

## ⚙️ Technical Architecture
- *Language:* Python 3.x
- *Frontend:* Streamlit Framework
- *Forensic Engine:* Custom Heuristic & Mathematical (Entropy) Algorithms
- *Documentation:* ReportLab (Professional PDF Generation)
