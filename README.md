🛡️ CyberSentinel | Forensic Neural OS
CyberSentinel is a local-first security engine built for real-time URL analysis and forensic threat assessment. By utilizing Shannon Entropy analysis and heuristic pattern detection, it provides instant, privacy-focused security intelligence without relying on external cloud APIs.
🎯 Mission
My mission with CyberSentinel is to provide a transparent, local, and user-centric security tool. In an era where many forensic tools risk data privacy by offloading analysis to the cloud, CyberSentinel ensures all calculations are performed locally, keeping your investigations secure and private.
🛠️ Core Features
 * Heuristic Analysis Engine: Detects malicious patterns (obfuscation, phishing keywords, and insecure protocols) in seconds.
 * Shannon Entropy Scoring: Uses mathematical models to quantify the risk of URL randomness.
 * Automated Forensic Reporting: Generates comprehensive PDF reports for security incidents with one click.
 * Local Data Sovereignty: All analysis stays on your machine; zero third-party data transmission.
⚙️ Technical Architecture
 * Language: Python
 * Interface: Streamlit (High-performance web dashboard)
 * Reporting: ReportLab (Professional-grade forensic documentation)
 * Intelligence: Custom heuristic algorithms and entropy modeling
